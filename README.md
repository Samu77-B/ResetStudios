# Reset Studios

Marketing site for **Reset Studios** — holistic health coaching and online personal training in the UK.

Live brand site: [resetstudios.co.uk](https://www.resetstudios.co.uk/)

## Stack

- Next.js (App Router)
- Tailwind CSS
- TypeScript

## Develop

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Assets

Web-optimised images live in `public/images` and logos in `public/logos`. Original source media is kept locally under `assets/` (not committed).
